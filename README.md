# Bio-OS 프리미엄 Console (Streamlit)

프리미엄 투자자용 콘솔 레이아웃(Zone 분리/테두리/타이틀바/카드)을 Streamlit에서 구현한 뼈대입니다.

## 실행
```bash
pip install -r requirements.txt
streamlit run app.py
```

## 연결 포인트
- `load_latest_metrics()`를 Seed-M1 DB/API 호출로 교체하면, KPI/Loop 비교/근거/증설 단계가 실제 값으로 출력됩니다.
- 내부 로직/파라미터는 변경하지 않고, 외부 표현(전문가 톤)만 담당합니다.


## 한글 표준 고정
- 화면에 영어 표기는 나오지 않도록 기본값을 한글로 고정했습니다.


## 용어 표준(현장용)
- 안정→정상, 경고→경계
- 리스크 지수→전체 위험 점수
- 급변 이벤트→갑작스런 변화
- 노출→위험 노출 시간
- 부담률/Utilization→설비 사용률
- P1/P2/P3는 화면에서 한글(지금 바로 조치/오늘 안에 점검/계획 수립 필요)로 표기


## 데모 시나리오
- 사이드바에서 '산소량 급락/물 흐름 저하/여과 부담 증가'를 선택하면 KPI/근거/조치가 시나리오에 맞게 변경됩니다.
- 내부 로직/파라미터는 변경하지 않고 화면 데모용 값만 바뀝니다.


## 보고서(PDF) 생성
- 사이드바 '보고서'에서 1페이지 요약/3페이지 상세를 선택하면 PDF를 생성하고 다운로드할 수 있습니다.
- ReportLab 기반이며 표준 한글 용어를 사용합니다.


## 제출 정보(정부/투자자)
- 사이드바 '제출 정보'에서 시설명/보고 기간/작성자/시스템 버전을 입력하면 PDF 상단에 자동 반영됩니다.


## 정부 제출용 고정 문구
- 3페이지 상세 보고서(Page 3)에 ‘사업 목적 및 기대효과’, ‘데이터 기반 의사결정 흐름(도식)’ 섹션이 자동 포함됩니다.


## 글로벌(영문) 보고서
- 현장 UI는 한글 고정입니다.
- 사이드바에서 '글로벌 보고서 표시'를 켜면 영문 1p/3p PDF를 생성할 수 있습니다.


## 영문 표준 용어 사전(EN Glossary)
- 영문 상세 PDF의 마지막 페이지에 한글(UI) ↔ 영문(Global) 용어 사전이 포함됩니다.


## 문서번호/개정/로고(텍스트) 지원
- 사이드바 '제출 정보'에 문서 번호, Rev., 발행기관/회사, 로고(텍스트)를 추가했습니다.
- PDF 상단 헤더/하단 푸터에 자동으로 들어갑니다.


## 개정 이력(Rev. History)
- 상세 보고서에 Rev. 표가 자동 생성됩니다.
